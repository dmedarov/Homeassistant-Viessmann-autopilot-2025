# Contributing to Viessmann Autopilot 2025

Thank you for your interest in improving the best open-source heat pump brain in the world!

We welcome all contributions: bug reports, feature requests, documentation improvements, translations, and code.

### How to contribute

#### 1. Report bugs or request features
- Go to [Issues](https://github.com/dmedarov/Homeassistant-Viessmann-autopilot-2025/issues)
- Search if the issue already exists
- Create a new issue with a clear title and description
- Use templates if available

#### 2. Suggest improvements or share your results
- Use **[Discussions → Ideas & Results](https://github.com/dmedarov/Homeassistant-Viessmann-autopilot-2025/discussions/categories/ideas-results)**  
  Perfect place for:
  - Your real-world SCOP / savings
  - Screenshots of your dashboard
  - Tuning tips for different Viessmann models or climates
  - Questions & answers

#### 3. Submit code (pull requests)
1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly (especially safety limits!)
5. Update documentation if needed
6. Open a Pull Request with a clear description

#### Code style
- Follow existing style (PEP8)
- Keep functions small and readable
- Always keep safety limits intact
- Add comments for complex logic

#### Testing
- Test on your own system first
- Do not increase supply temperature limits (>59 °C is dangerous)
- Keep the "no risk" philosophy

### Thank you!
Every contribution makes heating cheaper and greener for someone.  
You are helping families save real money and reduce CO₂.

Made with ❤️ in Bulgaria – for the whole world.
